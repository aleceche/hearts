# Hearts
An online room-based card game: Hearts

# Getting Started

## Prereqs

- Node.js: `10.3.0`
- NPM: `6.4.0`

## Setup

Clone this repository...
```
git clone git@github.com:darkterbear/hearts
cd hearts/
```

Install server dependencies and start the server
```
cd ./server
npm i
npm start
```

Install client dependencies and start React app
```
cd ./client
npm i
npm start
```

Use `localhost:3000` to access the application

# Built With
- Express.js
- React
- Socket.io and Socket.io Client
